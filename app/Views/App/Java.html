Java - Notifier Mobile Service

Download

You can download Notifier Mobile Service as a JAR file.

Download

Or

Using Maven to import the library to your project

<dependency>
    <groupId>com.notifiermobile</groupId>
    <artifactId>notifier-mobile-service</artifactId>
    <version>1.0</version>
</dependency>

Note: For others such as Gradle/Grails, Groovy Grape, Apache Ivy, Scala SBT, Leiningen ... you can get here

By Notifier Mobile Service, you can CREATE/GET/UPDATE/DELETE notifications easily just with a few lines of code.

Import library

    import com.notifiermobile.NotifierMobileService;

Create Notification

    // Create object for authentication
    Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

    // Create new notification
    Notification notification = new Notification();
    notification.setTitle("title");
    notification.setMessage("message");
    notification.setType(NotificationType.INFO.ordinal());

    // Add notification by service
    NotifierMobileService.add(notification, authentication);

Note: Notification id is generated automatically, you do not need to set id for new notification

Get All Notifications

    // Create object for authentication
    Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

    // Get all notifications by service
    NotifierMobileService.getAll(authentication);

Note: You can get notifications by type (INFO, WARNING, ERROR) or status of notification (unread or read)

Get Notification By Id

    // Create object for authentication
    Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

    // Get notification with id = 1 by service
    NotifierMobileService.get(1, authentication);

Update Notification

    // Create object for authentication
    Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

    Notification notification = new Notification();
    // Update message
    notification.setMessage("message");

    // Update message for notification with id = 1 by service
    NotifierMobileService.update(1, notification, authentication);

Note: You can not update id of notification

Mark Notification As Read

    // Create object for authentication
    Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

    // Mark notification with id = 1 as read by service
    NotifierMobileService.markAsRead(1, authentication);

Delete Notification

    // Create object for authentication
    Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

    // Delete notification with id = 1 by service
    NotifierMobileService.delete(1, authentication);

Exception

A NotificationException will be throwed when there is an error during connecting to server.

    try {
        // Create object for authentication
        Authentication authentication = new Authentication("yourUsername", "yourSecretKey");

        // Create new notification
        Notification notification = new Notification();
        notification.setTitle("title");
        notification.setMessage("message");
        notification.setType(NotificationType.INFO.ordinal());

        // Add notification by service
        NotifierMobileService.add(notification, authentication);
    } catch (NotificationExcception e) {
        System.out.println(e);
    }

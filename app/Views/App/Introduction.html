It's easy to create a notification with a few lines of code

You can do this with 2 libraries Java or .NET. For other languages, since Notification API is introduced, you can implement it by yourself easily. 


  Authentication authentication = new Authentication("yourUsername", "yourSecretKey");
  Notification notification = new Notification("title", "message", NotificationType.INFO.ordinal());

  NotifierMobileService.add(authentication, notification);

Get your notifications in real time with Notifier Mobile application

Maximum of 5 minutes as soon as the notifications have been raised by your code, Notifier Mobile application will get all of them automatically and notice you.

Notification - Get all notifications

Get all notifications by type or unread

http://notifiermobile.com/api/notifications?username={yourUsername}&secretkey={yourSecretKey}

Parameter

Field Type Description
type optional

Number

Optional Type of notification (0 = Info, 1 = Warning, 2 = Error).
unread optional

Boolean

Optional Get notifications that were read or not (true = unread, false = read).
fromId optional

Number

Optional Get notifications which id is greater than specified id.
  • Request-Example: 
{ 
    "Type": 0,
    "Unread": true
}

Success 200

Field Type Description
response

Notification[]

List of notifications.
  • Success-Response: 
[
     {
         "Id": 1,
         "Title": "title 1",
         "Message": "message 1",
         "Type": 0,
         "UnRead": true,
         "CreateDate": "2015-09-24T07:35:23.017"
     },
     {
         "Id": 2,
         "Title": "title 2",
         "Message": "mesage 2",
         "Type": 1,
         "UnRead": false,
         "CreateDate": "2015-09-24T07:26:53.5"
     }
 ]

Error 4xx

Field Description
401

Unauthorized (The username or secretKey is invalid).
400

Bad request (The parameter is invalid).
500

Internal server error.

Notification - Get notification by Id

Get notification by id

http://notifiermobile.com/api/notifications/:id?username={yourUsername}&secretkey={yourSecretKey}

Parameter

Field Type Description
id

Number

Id of notification.

Success 200

Field Type Description
response

Notification

Notification with specified id.
  • Success-Response: 
{
     "Id": 1,
     "Title": "title 1",
     "Message": "message 1",
     "Type": 0,
     "UnRead": true,
     "CreateDate": "2015-09-24T07:35:23.017"
 }

Error 4xx

Field Description
401

Unauthorized (The username or secretKey is invalid).
400

Bad request (The parameter is invalid).
404

Not found (The notification with specified id does not exist).
500

Internal server error.

Notification - Create new notification

Create new notification

http://notifiermobile.com/api/notifications?username={yourUsername}&secretkey={yourSecretKey}

Header

Field Description
Content-Type

application/json

Parameter

Field Type Description
title

String

Title of notification.

Max length: 200
message optional

String

Optional Content of notification.

Max length: 500
type optional

Number

Optional Type of notification (0 = Info, 1 = Warning, 2 = Error).
createDate optional

CreateDate

Optional Created Date of notification.
  • Request-Example: 
{ 
    "Title": "title",
    "Message": "message",
    "Type": 0
}
  • Success-Response: 
HTTP/1.1 201 Created

Error 4xx

Field Description
401

Unauthorized (The username or secretKey is invalid).
400

Bad request (The parameter is invalid).
403

Forbidden (You reached to limit of notifications a day).
500

Internal server error.
